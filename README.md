# 🍛 Punjabi Khana - Restaurant Landing Page

A fully responsive, single-page website built for **Punjabi Khana**, an authentic North Indian vegetarian restaurant located in Nehru Place, New Delhi. 

This project serves as a modern landing page designed to attract local customers, showcase the restaurant's signature dishes and full menu, and drive delivery orders via direct call or delivery partners (Swiggy/Zomato).

## ✨ Features

* **Responsive Design:** Fluid layout that adapts perfectly to desktop, tablet, and mobile devices.
* **Sticky Mobile CTA Bar:** A fixed bottom bar on mobile screens for quick access to calling and ordering.
* **Delivery Integration Banner:** A top banner highlighting availability on Zomato and Swiggy.
* **Full Digital Menu:** A clean, two-column responsive grid displaying the complete restaurant menu with prices.
* **Interactive Navigation:** Smooth scrolling links and a mobile-friendly hamburger menu.
* **Location Integration:** Embedded interactive Google Maps for easy navigation to the restaurant.

## 🛠️ Technologies Used

* **HTML5:** Semantic markup for structure and accessibility.
* **CSS3:** Custom styling using CSS variables, Flexbox, and Grid for layouts. (No external CSS frameworks used).
* **JavaScript (Vanilla):** Minimal, lightweight script for toggling the mobile navigation menu.
* **Google Fonts:** Utilizing 'Poppins' for body text and 'Alfa Slab One' for display headings.
